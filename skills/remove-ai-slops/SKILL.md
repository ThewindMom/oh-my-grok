# Remove AI Slop

AI-generated code often contains patterns that add no value and signal machine generation. Remove them.

## Comment patterns to remove

1. **Praise comments**: `// Great function!`, `// Beautiful implementation`
2. **Restating comments**: `// This function returns the sum of two integers`
3. **Attribution comments**: `// Generated by AI`, `// Created by ChatGPT`
4. **End-of-block comments**: `// end of function`, `// end of if`
5. **Filler comments**: `// Note: this is important`, `// Don't forget to...`
6. **Helper labels**: `// This is a helper function`

## Code patterns to remove

1. **Unnecessary wrappers**: functions that just call another function without adding logic
2. **Over-engineered abstractions**: interfaces with one implementation, factories for simple types
3. **Unused imports**: AI often adds imports "just in case"
4. **Dead code**: variables declared but never used
5. **Redundant nil checks**: checking for nil when the type guarantees non-nil
6. **Unnecessary type assertions**: asserting a type that's already known

## Process

1. Scan all changed files for the patterns above.
2. Remove each instance.
3. Verify the code still compiles and tests pass.
4. Commit the cleanup separately from the feature change.

## Rules

- Don't remove comments that provide genuine context (RFC references, design decisions, non-obvious behavior)
- Don't remove comments that explain WHY, only those that explain WHAT
- Don't change behavior while removing slop
- Run tests after cleanup to verify no breakage
